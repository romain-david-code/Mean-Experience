Pile technologique (stack)

La stack MEAN regroupe un ensemble de technologies JavaScript (et/ou TypeScript) permettant de créer des projets full stack. Elle se compose de 4 technologies principales:

  • MongoDB
  • Express
  • Angular
  • Node.js
MongoDB

MongoDB est la base de donnée du stack. Ce système est NoSQL (non-relationel). C'est-à-dire que les données sont stockées sous forme de documents JSON donc indépendants les un des autres, à la différence des bases de données SQL.

Express

Express est un framework permettant de créer un serveur web (backend). Il assure la liaison entre l'interface utilisateur et MongoDB (via Mongoose par exemple).

Angular

Angular est un framewrok frontend basé sur MVC (Model View Controller). Il permet de définir un interface utilisateur à partir de composants (classes) réutilisable.

Node.js

Node.js est un environnement JavaScript permettant d'exécuter du code sur un serveur. Il bénificie d'un répertoire de modules complémentaires (npm) offrant une multitude de fonctionnalités (tel que Express et Mongoose).

Intérêt de la stack MEAN

Cette pile technologique permet d'utiliser uniquement des technologies JavaScript ce qui implique une meilleure compatibilité entre les outils de développement.

La stack MEAN est très proche de MERN (React) et MEVN (Vue.js). On peut donc se poser la question du choix d'utiliser Angular, moins populaire.

Angular se démarque sur des projets avec une complexitée importante grâce à son système de composant. De plus, il utilise maintenant TypeScript qui est de plus en plus populaire et est un avantage non négligeable afin de détecter au plus tôt des bugs pendant la phase de développement.
